## Menu and User authority management web application
### 메뉴 및 사용자 권한처리 수행 어드민페이지
Spring Security 미적용 세션, 인터셉터 활용 접근 및 저장 로직 권한 처리

### 기능
- 로그인
- 메뉴 등록/수정/삭제 (권한 처리) : 조회 권한, 등록/수정/삭제 권한
- 관리자 등록/수정/삭제 (권한 처리)
- 메뉴 권한 일괄 수정

### 사용기술
- Spring Boot 3.2.x
- JPA(Querydsl)
- Thymeleaf
